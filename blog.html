Question-1: What are the different ways to select an element in the DOM?
We can select an element with getElementsById,getElementsByClassName, getElementsByTagName, querySelector and querySelectorAll.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
Using innerHTML can add tag element like h1, div etc. innerText we can add value inside the tag. textContent get or set the text content.
Question-3: What is event delegation in the DOM?
Event delegation helps to write less code and once we create a function we can use many times.
Question-4: What is event bubbling in the DOM?
Event bubbling find start from document and find the trigger point, when find the trigger point it return the start point
Question-5: How do you create, add, and remove elements using JavaScript?
We can create element using createElement, add using appendChild and remove element using remove.
